#  Bikes Sales Dashboard (Excel)

##  Overview
This project features an Excel dashboard that provides insights into bike sales performance across different regions, products, and customer segments.

##  Key Insights
- Mountain-200 is the best-performing product  
- The United States generates the highest revenue  
- Sales vary significantly by product size  
- Male customers contribute more to total revenue  

##  Features
- KPI Cards (Total Sales, Total Profit, Customer Count)  
- Sales analysis by:
  - Country  
  - Gender  
  - Product Type  
  - Size  
- Top-performing regions and states  
- Interactive slicers for filtering
